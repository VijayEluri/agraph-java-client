Low-level communication with the AllegroGraph server.

These classes implement low-level communication with the AllegroGraph server. For proper modularity, code outside this package must only reference members of the AGConnector class.

The only public members in classes other than AGConnector are the implementatations of the abstract methods of AGConnector.

Disclaimer

The public declarations are for internal use only. References to the members of these classes are not supported in application programs.
